Placez ici les fichiers de police Roboto Slab (format recommandé: .woff2).

Noms de fichiers attendus par la configuration CSS:
- RobotoSlab-Light.woff2    (weight 300)
- RobotoSlab-Regular.woff2  (weight 400)
- RobotoSlab-Medium.woff2   (weight 500)
- RobotoSlab-Bold.woff2     (weight 700)

Comment obtenir les fichiers:
1. Allez sur https://fonts.google.com/specimen/Roboto+Slab
2. Sélectionnez les graisses 300, 400, 500, 700 puis cliquez sur "Download family".
3. Décompressez l'archive et copiez les fichiers .woff2 adaptés dans ce dossier en les renommant si nécessaire pour correspondre aux noms ci-dessus.

Astuce pour vérifier en local:
- Lancez votre app (`npm run dev`) et ouvrez le panneau Réseau (Network) dans les outils dev du navigateur, filtrez sur "font" pour vérifier que les fichiers se chargent depuis `/fonts/`.

Si vous voulez, je peux renommer automatiquement les fichiers si vous les ajoutez dans un autre dossier — dites-moi où vous les placerez.
